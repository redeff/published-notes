Ejercicio 1

Dado el conjunto \(A = \{(x, y) \in \mathbb{R}^2 : -x < y, y + x^2 - 2 \leq 0\}\) No es abierto ya que el punto \((0, 2)\) está en \(A\) pero \((0, 2 + \varepsilon) \notin A\) para todo \(\varepsilon> 0\).

Notemos que tampoco es cerrado, ya que \((0, 0) \notin A\), pero \((0, \varepsilon) \in A\) para cualquier \(\varepsilon\) suficientemente chico.

Conjuntos Acotados

Un conjunto \(A \subseteq\mathbb{R}^n\) es acotado si existe alguna bola \(A \subseteq B_M(0)\)

Propiedades

  • \(A^\circ \subseteq A\)

  • \(A^\circ\) es abierto

  • si \(A\) abierto luego \(A^\circ = A\)

Punto de Acumulación (Clausura)

Un punto \(x \in \mathbb{R}^n\) es un punto de acumulación si existe una sucesión \(a : \mathbb{N}\to A\) tal que \(\lim\limits_{i \to \infty} ai = x\).

Se define al conjunto de puntos de acumulación como \(\mathop{\mathrm{cl}}A = \overline A\). Es claro que \(A \subseteq\overline A\), y además \(\overline A = \mathop{\mathrm{cl}}A\) es cerrado.

Borde

el borde de \(A\) se define como \(\partial A = \mathop{\mathrm{cl}}A - A^\circ\).

Alternativamente, \(\partial A = \{ x : \forall \varepsilon> 0 : B_r(x) \cap A \neq \emptyset, B_r(x) \cap A^C \neq \emptyset \}\)

Ejercicio 2

para los siguientes conjuntos, decidir si son acotados, y hallar el interior, la clausura y el borde.

  • \(A = [-1, 2)\)

  • \(B = \{(x, y) \in \mathbb{R}^2 : 1 < (x-1)^2 + y^2 < 4\}\)

  • \(C = \{(x, y) : |x| \leq 3\}\)

Cónicas

Una cónica es una es una curva en \(\mathbb{R}^2\) que surge de cortar un cono con un plano. Se dividen en:

  • Circunferencia: \(x^2 + y^2 = r^2\).

  • Elipse: \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\)

  • Parábola: \(y = x^2\)

  • Hipérbola: \(\frac{x^2}{a^2} - \frac{x^2}{b^2} = 1\)
