Dado \(S \in \mathbb{V}\), \(\langle S \rangle\) es la combinación. Se dice que \(S\) es un sistema de generadores de \(\langle S \rangle\) lineal de finitos de ellos.

Tenemos que son exactamente los \(X\) tal que el sist \((v_i \mid X)\) tiene solución, entonces triangulamos y le pedimos que sea compatible.

Dependencia Lineal

Un conjunto \(X\) es ld si existe alguna combinación lineal no trivial de vectores en \(X\) que de \(0\)

Ejemplo

Dado \(\mathbb{R}^\mathbb{N}\) y la base canónica \(e_i \; n = n == i\), tenemos que \(\{e_i\} \cap \text{ const } 1\) es li

Supongamos que no, luego \(\sum\limits\lambda_i x_i = 0\), si todos los \(x_i\) son de la canónica, ya estamos Si no, wlog \(x_1 = \text{ const } 1\), y tenemos que a partir de un momento \(\left(\sum\limits\lambda_i x_i\right) n\) es siempre \(\lambda_1\) para \(n\) suficientemente grande, luego \(\lambda_1 = 0\) y volvemos al caso anterior

Bases

Dado un \(\mathbb{V}\) y un subconjunto \(S \subseteq \mathbb{V}\) finito, decimos que \(S\) es base iff \(\langle S \rangle = \mathbb{V}\) y \(S\) es li

Decimos que \(\dim \mathbb{V}= |S|\)

Prop

si \(\dim \mathbb{V}= n\) y \(|S| = n\), tenemos:

\(S\) base iff \(\langle S \rangle = \mathbb{V}\) iff \(S\) li

Extender a una Base

Si tenemos un conjunto de vectores \(S \subseteq \mathbb{R}^n\), luego triangulamos la matriz \(\begin{bmatrix} v_i \\ \vdots \\ e_i \end{bmatrix}\)

Extender a Bases de otros Conjuntos

Extender \(\{(1 \; -1 \; 0 \; 1) \; (0 \; 1\; 0 \; -1)\}\) a una base de \(T = \{x \in \mathbb{R}^4 \mid x_2 + x_4 = 0\}\).

Notemos que \((0 \; 0\; 1\;0)\) está en \(T\) y es li con los que tenemos, entonces extenderla con ele alcanza

Ejercicio 3

Hallar un conjunto de generadores finit y una base de \(S = \{P \in \mathbb{R}[x]_{\leq 5} \mid P(0) = P(2) = 0, P(1) = P(-1) \}\)

Tenemos \(P(x) = x(x-2)(a_0 + a_1x + a_2x^2 + a_3x^3)\)

\(P(1) = -a_0 - a_1 - a_2 - a_3\)

\(P(-1) = 3a_0 - 3a_1 + 3a_2 - 3a_3\)

Luego \(-2 a_0 + a_1 - 2_a2 + a_3 = 0\)

\(a_1 = 2a_0 + 2a_2 - a_3\)

Y a partir de eso armamos los generadores
