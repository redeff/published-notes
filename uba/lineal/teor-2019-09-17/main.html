Anulador

Dado \(S\) un subespacio de \(\mathbb{V}\), si definimos \(T = \{x : \mathbb{V}: \forall \phi \in S^0 : \phi x = 0\}\), luego \(S = T\).

Es claro que \(S \subseteq T\). Vemos que \(T \subseteq S\). Sea \(x \in T\), \(x \notin S\), tomemos \(v\) una base de \(\mathbb{V}\), con \(v_{i < r}\) base de \(S\), y \(v_r = x\). Sea la base dual \(\phi = v^*\). Como \(\phi_{i \geq r}v_{i < r} = 0\), tenemos que \(\phi_{i \geq r}\) es base de \(S^0\).

Luego \(\phi_r \subseteq S^0\), entonces \(\phi_r x = 0\), pero \(x = v_r\), pero entonces por definición de base dual tenemos \(\phi_r x = 1\). Contradicción.

Observación

Dado \(\mathbb{V}\) finitamente generado y \(S\) sub de \(\mathbb{V}\), lugo si \(\phi\) es base de \(S^0\), luego \(S = \{x \in \mathbb{V}: \forall i : \phi_ix = 0\}\).

Doble dual

Tenemos la función \(\mathop{\mathrm{ev}}\in (\mathbb{V}\to \mathbb{V}^{**} = \mathbb{V}\to \mathbb{V}^{*} \to \mathbb{K})\) dada por \(\mathop{\mathrm{ev}}_x \phi = \phi x\). La función \(\mathop{\mathrm{ev}}\) también se llama \(\Phi\).

Esta función es \(\mathop{\mathrm{iso}}\) para \(\mathbb{V}\) finitamente generado.

Anulador del Anulador

Tenemos \(S^{00} = \{\Phi : \mathbb{V}^{**} : \forall \phi \in S^0 : \Phi\phi = 0\}\). Por lo anterior, tenemos que \[S^{00} = \{\Phi \in \mathbb{V}^{**} : \forall \phi \in S^0 : \Phi\phi = 0\} =\] \[\{\mathop{\mathrm{ev}}_x : x \in \mathbb{V}: \forall \phi \in S^0 : \mathop{\mathrm{ev}}_x\phi = 0\} =\] \[\{\mathop{\mathrm{ev}}_x : x \in \mathbb{V}: \forall \phi \in S^0 : \phi x = 0\} =_{\mathop{\mathrm{iso}}}\] \[\{x : x \in \mathbb{V}: \forall \phi \in S^0 : \phi x = 0\} = S\] Luego tenemos \(S^{00} =_{\mathop{\mathrm{iso}}} S\)

Operaciones de Subespacios sobre Anulación

Se cumple que:

  • \((S+T)^0 = S^0 \cap T^0\)

  • \((S\cap T)^0 \subseteq^T S^0 + T^0\)

Y para dimensión finita,

  • \((S+T)^0 = S^0 \cap T^0\)

  • \((S\cap T)^0 = S^0 + T^0\)

Función Transpuesta

Dada \(f : \mathbb{V}\to \mathbb{W}\), la función transpuesta \(f^t : \mathbb{W}^* \to \mathbb{V}^*\) es la dada por \(f^t \phi = \phi \circ f\), es decir \(f^t = (\circ f)\). notemos que \((*^t) : (\mathbb{V}\to \mathbb{W}) \to (\mathbb{W}^* \to \mathbb{V}^*)\) es lineal. Esto es ya que:

  • \(f^t \phi\) es lineal.

  • \(f^t\) es lineal

  • \((*^t)\) es lineal

Transpuesta de la Identidad

Dada \(\mathop{\mathrm{id}}: \mathbb{V}\to \mathbb{V}\), tenemos que \(\mathop{\mathrm{id}}^t = \mathop{\mathrm{id}}_{\mathbb{V}^*}\)

Composición

Dada \(f : \mathbb{U}\to \mathbb{V}\) y \(g : \mathbb{V}\to \mathbb{W}\), tenemos que \((g \circ f)^t = f^t \circ g^t : \mathbb{W}^* \to \mathbb{U}^*\).

Demo: \[(g \circ f)^t \phi\] \[= \phi \circ (g \circ f)\] \[= (\phi \circ g) \circ f\] \[= g^t \phi \circ f\] \[= f^t (g^t \phi)\] \[= (f^t \circ g^t) \phi\]

Isomorfismo

Si \(f : \mathbb{V}\to_{\mathop{\mathrm{iso}}} \mathbb{W}\), luego \(f^t : \mathbb{W}\to_{\mathop{\mathrm{iso}}} \mathbb{V}\), y además \((f^{-1})^t = (f^t)^{-1}\)

Notemos que \[f^t \circ (f^{-1})^t = (f^{-1} \circ f)^t = \mathop{\mathrm{id}}^t = \mathop{\mathrm{id}}\] y similarmente, \[(f^{-1})^t \circ f^t = (f \circ f^{-1})^t = \mathop{\mathrm{id}}^t = \mathop{\mathrm{id}}\]

Anuladores

Si \(f: \mathbb{V}\to \mathbb{W}\), luego \(\ker f^t = (\mathop{\mathrm{Im}}f)^0\). Además, \(\mathop{\mathrm{Im}}f^t \subseteq(\ker f)^0\), la igualdad dándose para dimensión finita.

Notemos que \[(\mathop{\mathrm{Im}}f)^0 = \{\phi : \phi (\mathop{\mathrm{Im}}f) = 0\} =\] \[\{\phi : (\phi \circ f) \mathbb{V}= 0\}\] \[\{\phi : f^t \phi \mathbb{V}= 0\}\] \[\{\phi : f^t \phi = 0\} = \ker f^t\]

Supongamos \(f^t \psi = \phi\), tenemos \[f^t \psi = \phi\] \[\psi \circ f = \phi\] \[(\psi \circ f) \ker f = \phi \ker f\] \[\psi (f \ker f) = \phi \ker f\] \[\psi 0 = \phi \ker f\] \[0 = \phi \ker f \Rightarrow\phi \subseteq(\ker f) ^0\]
