Expresar en Base

Dado \(v \in \mathbb{V}\), y una base \(\mathcal{B}_\mathbb{V}\), tenemos que \((v)_{\mathcal{B}_\mathbb{V}} = (\lambda_i) \iff \sum\limits\mathcal{B}_{\mathbb{V}_i} \cdot \lambda_i = v\)

Igualdad

Si tenemos \(f, g \in \mathop{\mathrm{Hom}}\mathbb{V}\; \mathbb{W}\), luego \(f = g \iff [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}} = [g]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}}\)

Composición

Si \(f : \mathbb{U}\to \mathbb{V}\) y \(g : \mathbb{V}\to \mathbb{W}\), luego \([g \circ f] = [g] \times [f]\)

Corolario

Tenemos \(\mathbb{V}\), \(\mathbb{W}\) de dimensiones finitas respectivamente; y \(f \in \mathop{\mathrm{Hom}}\mathbb{V} \; \mathbb{W}\). Luego \(f \mathop{\mathrm{iso}}\iff [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}}\) es inversible.

Cambio de Base

Tomemos dos bases \(\mathcal{B}_\mathbb{V}\) y \(\mathcal{B}_\mathbb{V}'\) de \(\mathbb{V}\), con \(\dim \mathbb{V}= n\), y lo mismo para \(\mathbb{W}\) de dimensión \(m\). Tenemos que: \[[f]_{\mathcal{B}'_\mathbb{V}\mathcal{B}'_\mathbb{W}} = C(\mathcal{B}_\mathbb{W}, \mathcal{B}'_\mathbb{W}) \times [f]_{\mathcal{B}_\mathbb{V}\mathcal{B}_\mathbb{W}} \times C(\mathcal{B}_\mathbb{V}, \mathcal{B}'_\mathbb{V})^{-1}\]

Donde \(C(\mathcal{A}, \mathcal{B}) = [\mathop{\mathrm{id}}]_{\mathcal{A}\mathcal{B}}\).

Definición

Dados \(A, B \in \mathbb{K}^{n \times n}\), decimos que \(A, B\) son semejantes sii: \[A \sim B \iff \exists C \in \mathop{\mathrm{GL}}\mathbb{K}^n : A = C \times B \times C^{-1}\]

Interpretación de Semejanza

Decimos que \(A \sim B \iff \exists f \in \mathop{\mathrm{Hom}}\mathbb{K}^n \; \mathbb{K}^n :\exists \mathcal{B}, \mathcal{B}' : [f]_\mathcal{B}= A, [f]_{\mathcal{B}'} = B\).

Demo: Trivial por cambio de base con \(\mathbb{V}= \mathbb{W}\).

Rango de una Matriz

Si tenemos una matriz \(A \in \mathbb{K}^{n \times m}\), y una \(f\) tal que \([f] = A\), luego decimos que: \(\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}}[f] = \dim \mathop{\mathrm{Im}}f\)

Dimensión de Soluciones de un Homogéneo

Si tenemos \(A \in \mathbb{K}^{m \times n}\), luego tenemos \(\dim \{x : Ax = 0\} = n - \mathop{\mathrm{rg}}A\).

Sistemas no Homogéneos

Si tenemos el sistema lineal \(Ax = b\), entonces tiene solución sii \(\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}} (A \mid b)\).

Notemos que \(\mathop{\mathrm{rg}}A = \mathop{\mathrm{rg}}(A \mid b) \iff \dim \mathop{\mathrm{Im}}A = \dim \mathop{\mathrm{Im}} (A | b) \iff \dim \langle A_{*i} \rangle = \dim (\langle A_{*i} \rangle + \langle b \rangle) \iff\) b es combinación lineal de las columnas de \(A \iff Ax = b\) tiene solución.

Matrices Inversibles

Demostremos que dada \(A \in \mathbb{K}^{n \times n}\), luego \(A \times B = I \Rightarrow B \times A = I\).

Tomemos \(f : \mathbb{K}^{n \times n} \to \mathbb{K}^{n \times n}, fX = AX\). Luego notemos que \(\mathop{\mathrm{epi}}f\), ya que \(f(B \times C) = A \times (B \times C) = (A \times B) \times C = C\).

Luego por dimensión, tenemos \(\mathop{\mathrm{iso}}f\). Pero además \(f(B \times A - I) = A \times (B \times A - I) = A \times B \times A - A = I \times A - A = 0\), luego \(B \times A = I\)

Sistemas de Ecuaciones

  1. \(A\) inversible

  2. \(Ax = 0\) tiene solución única

  3. \(Ax = b\) tiene única solución

  4. \(Ax = b\) tiene al menos una solución

  5. \(\mathop{\mathrm{rg}}A = n\)
