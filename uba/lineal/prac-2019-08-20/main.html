Rec

\(B \subseteq \mathbb{V}\) es base \(\iff\) \(B\) es li y \(\langle B \rangle = \mathbb{V}\)

Ejercicio 1

Dado \(\mathbb{V}= \mathbb{K}_n[x] = \{p \in \mathbb{K}[x] \mid p = 0 \text{ ó } \deg(p) \leq n\}\).

Si \(B = \{p \mid \deg(p) = i | 0 \leq i \leq n\}\), demostrar que \(B\) es base.

Demo 1

La matriz en \(E = \{x^i \mid 0 \leq i \leq n\}\) es triangular.

Demo 2

Inducción. \(n = 0\) es trivial.

Si \(n \geq 1\), supongamos \(\sum\limits\lambda_iB_i = 0\). El coeficiente \(n\) de ese polinomio tiene que ser cero, luego \(\lambda_n = 0\), luego \(0 = \sum\limits_{i < n} \lambda_iB_i = 0\), pero esto no puede pasar por la hipótesis inductiva.

Ejercicio 8

Tomemos \(S = \left\{A \in \mathbb{K}^{n \times n} \; \middle| \; \sum\limits_j A_{ij} = 0 = \sum\limits_i A_{ij}\right\}\)

Calcular \(\dim S\). Notemos que si dejamos libre los \(A_{ij}\) con \(i < n\) y \(j < n\) y tomamos \(A_{nj} = -\sum\limits_{i < n} A_{ij}\), y \(A_{in} = -\sum\limits_{j < n} A_{ij}\), y \(A_{nn} = -\sum\limits_{i,j < n} A_{ij}\), esta matriz cumple, y son todas las que cumplen.

Luego da \(\dim S = (n-1)^2\)

Intersección y suma de subespacios

Dados \(S\), \(T\) sub. de \(\mathbb{V}\), se tiene

  • \(S \cap T = S \cap T\)

  • \(S + T = \{s + t \mid s \in S, t \in T\}\)

  • \(S \oplus T = S + T\) si están en suma directa, que se da cuando:

    • \(S \cap T = {0} \iff\)

    • \(\dim (S + T) = \dim S + \dim T \iff\)

    • \(\forall v \in S + T \; \exists! \; s \in S, t\in T \mid s + t = v\)

Complemento

Cuando \(S \oplus T = \mathbb{V}\), decimos que \(S\) es complemento de \(T\). Se sabe que eso implica \(\dim S = n - \dim T\), es decir la dimensión de los complementos es toda la misma
